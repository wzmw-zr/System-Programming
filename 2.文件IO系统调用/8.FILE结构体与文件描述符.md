# FILE结构体与文件描述符

FILE结构体是库函数在用户空间定义的结构体，这个结构体中的成员大致如下：

```c
struct FILE
{
    char *_ptr;//文件输入的下一个位置
    int _cnt;//当前缓冲区的相对位置
    char *_base;//指基础位置（文件的起始位置）
    int _flag;//文件标志
    int _fileno;//文件描述符
    int _charbuf;//检查缓冲区状况，如果缓冲区则不读取
    int _bufsiz;//文件的大小
    char *_tmpfname;//临时文件名
};
```

**FILE结构体中是包含着文件描述符的。**
